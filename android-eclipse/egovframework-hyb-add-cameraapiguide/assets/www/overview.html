모바일 Camera 디바이스 API 가이드

설명

행정안전부 전자정부 표준프레임워크 센터에서는 전자정부 표준프레임워크를 이용하여 하이브리드 앱 개발 을 할 수 있는 모바일 디바이스 API 실행환경, 개발환경, 가이드프로그램을 제작 배포하고 있습니다.

Camera 디바이스 API 가이드 프로그램은 디바이스 기기 내 Camera와 갤러리를 이용하여 필요한 정보를 서버에 전송하는 기능 구현 시 참고 및 활용할 수 있는 가이드 앱 입니다.

기능사양

  • 적용 오픈 소스
    Cordova v8.1.2, jQuery Mobile 1.4.5, jQuery 2.1.4, eGovMoible 1.4.6, iScroll5, three.js
  • 적용 라이브러리
    android.jar, annotation.jar, spring-android-rest-template-1.0.1.RELEASE.jar, spring-android-core-1.0.1.RELEASE.jar
  • 운영 플랫폼
    Android SDK Revision 23 이상, 전자정부 표준프레임워크 3.9
  • 개발환경
    전자정부 표준프레임워크 개발환경 3.9
  • 적용 디바이스 API
    1. Cordova : Camera Device API, File ReadWriter Device API
    2. 전자정부 표준프레임워크 : Interface Device API

기능설명

  • 촬영 사진을 서버에 전송하기: 디바이스 API를 통해 카메라 장치 기능을 호출하여 사진을 촬영한 후 촬영된 사진을 화면에 표시하고 서버로 업로드하여 저장합니다.
  • 갤러리의 사진을 서버에 전송하기: 디바이스 API를 통해 갤러리에 저장된 사진을 화면에 표시하고 서버에 업로드하여 저장합니다.
  • 사진 목록 서버로부터 수신하기: 서버에 업로드한 사진 목록을 조회합니다.
