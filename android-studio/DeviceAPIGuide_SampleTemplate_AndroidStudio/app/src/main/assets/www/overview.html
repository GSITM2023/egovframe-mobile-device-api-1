모바일 표준 샘플 템플릿

설명

행정안전부 전자정부 표준프레임워크 센터에서는 전자정부 표준프레임워크를 이용하여 하이브리드 앱 개발 을 할 수 있는 모바일 디바이스 API 실행환경, 개발환경, 가이드프로그램을 제작 배포하고 있습니다.

모바일 디바이스 실행환경 표준 템플릿은 모바일 디바이스 API 실행환경을 활용하여 하이브리드 앱을 개발하시는 분들이 구현 시 참고 및 활용될 수 있도록 핵심 디바이스 API에 대한 샘플템플릿 앱 입니다.

기능사양

  • 적용 오픈 소스 : Cordova 8.1.2, jQuery Mobile 1.4.5, jQuery2.1.4, eGovMobile 1.4.6, iScroll5
  • 적용 라이브러리 : spring-android-rest-template-1.0.1.RELEASE.jar, spring-android-core-1.0.1.RELEASE.jar
  • 적용 디바이스 API :
    1. Cordova : Accelerometer, Camera, Capture, Compass, Connection, Contacts, Device, Events, File, Media, Notification, Geolocation, Storage
    2. 전자정부표준프레임워크 : 없음

기능설명

  • Accelerator : 디바이스 API를 통해 디바이스 모션 센서 접근(현재 가속도 정보 등등)정보를 호출하여, 모니터링 창에 표시합니다.
  • GPS: 디바이스 API를 통해 geoLocation 센서 접근(현재 좌표) 정보를 호출하여, 모니터링 창에 표시합니다.
  • Vibrator: 디바이스 API를 통해 단말을 진동시키고 모니터링 창에 표시합니다.
  • Camera: 디바이스 API를 통해 카메라 장치 접근 사진을 촬영하여 촬영된 사진을 화면에 표시합니다.
  • Media: 디바이스 API를 통해 사운드 장치 접근 음악을 재생합니다.
  • Contacts: 디바이스 API를 통해 현재 저장된 전화 번호부를 검색하여 화면에 표시합니다.
  • Compass: 디바이스 API를 통해 방향 센서 접근(방위각 정보) 정보를 호출하여 모니터링 창에 표시합니다.
  • FileReaderWriter: 디바이스 API를 통해 파일 익스플로러를 활성화여 선택된 파일을 모니터링 창에 표시합니다.
  • Network Info: 디바이스 API를 통해 네트워크 정보를 조회하여 모니터링 창에 표시합니다.
  • Device Info: 디바이스 API를 통해 디바이스 정보 를 조회하여 화면에 표시합니다.
