파일관리 디바이스 API 가이드

설명

행정안전부 전자정부 표준프레임워크 센터에서는 전자정부 표준프레임워크를 이용하여 하이브리드 앱 개발을 할 수 있는 모바일 디바이스 API 실행환경, 개발환경, 가이드프로그램을 제작 배포하고 있습니다.

파일관리 디바이스 API 가이드 프로그램은 모바일 디바이스 API 실행환경을 활용하여 하이브리드 앱을 개발하시는 분들이 디바이스 기기 내 파일 시스템을 조회하여 디렉토리 및 파일을 컨트롤하는 기능을 구현하는 데 참고 및 활용 할 수 있는 가이드 앱 입니다.

기능사양

  • 적용 오픈 소스
    Cordova v8.1.2, jQuery Mobile 1.4.5, jQuery 2.1.4, eGovMobile 1.4.6, iScroll5, Three.js
  • 적용 라이브러리
    CFNetwork.framework, Security.framework, libz.dylib
  • 운영 플랫폼
    iOS SDK 9 이상, 전자정부 표준프레임워크 3.9
  • 개발환경
    Xcode 10.3,
    전자정부 표준프레임워크 개발환경 3.9
  • 적용 디바이스 API
    1. PhoneGap : File Device API
    2. 전자정부 표준프레임워크 : Interface API

기능설명

  • 파일 Explorer : 모바일 디바이스의 디렉토리 및 파일을 조회합니다.
  • 디렉토리 생성/삭제 : 모바일 디바이스에 디렉토리 생성/삭제를 합니다.
  • 파일 복사/이동/삭제 : 모바일 디바이스의 파일을 복사/이동/삭제를 합니다.
